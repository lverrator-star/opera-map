# 《教我如何不想她》数字伴生作品 — 使用说明书

---

## 一、项目简介

《教我如何不想她》是一部关于中国现代语言学家、文学家、新文化运动先驱 **刘半农**（1891–1934）生平事迹的原创歌剧。刘半农是「她」字的创造者、中国实验语音学奠基人、新文化运动主将，与蔡元培、鲁迅、赵元任等人共同推动了中国现代学术的诞生。

**编剧**：彭峰、张吉义　　**作曲**：禹永一

本网页是该歌剧的 **「数字伴生作品」**——以交互式世界/中国双视图地图为核心，将刘半农在江阴、北京、伦敦、巴黎、绥远等 11 个地点的求学、执教、留学、采风经历串联起来，配合 10 场歌剧脚本内容与完整音频，形成一幅沉浸式的「生平地理叙事」。

## 三、快速启动

### Windows 用户（无需安装任何软件）

1. 解压 `operamap.zip` 到任意文件夹
2. 双击 **`start.bat`**
3. 浏览器会自动打开 → `http://localhost:8000`

> 💡 Windows 压缩包内已自带 **Python 3.12 嵌入式版**（位于 `runtime/python/`），无需单独安装 Python。

### Mac / Linux 用户

Mac 和 Linux 通常已预装 Python 3。如未安装：

```bash
# Mac
brew install python3

# Ubuntu / Debian
sudo apt install python3
```

启动：

```bash
cd operamap
chmod +x start.sh
./start.sh
```

浏览器会自动打开。若未自动打开，手动访问 `http://localhost:8000`。

### 手动启动（备选）

```bash
cd operamap
python server_nocache.py
```

然后浏览器打开 `http://localhost:8000`。

### 停止服务器

- **Windows**：关闭弹出的命令行窗口
- **Mac/Linux**：按 `Ctrl + C`

## 五、功能详解

### 5.1 双视图地图

点击顶部 **「寰宇」** 或 **「神州」** 切换地图视图。

| 视图 | 中心 | 显示内容 | 适用场景 |
|------|------|----------|----------|
| 寰宇 | 欧亚大陆 | 全部 11 个地点（中英法） | 纵观刘半农欧陆留学之路 |
| 神州 | 中国 | 国内 5 个地点 | 聚焦故乡、北大、绥远采风 |

地图上 **彩色圆形标记** = 地点节点。不同颜色代表不同人生阶段：

| 颜色 | 类别 | 含义 |
|------|------|------|
| 褐色 `●` | 故乡与求学 | 江阴 |
| 朱红 `●` | 执教 | 北京大学 |
| 靛蓝 `●` | 留学 | 伦敦、巴黎 |
| 灰绿 `●` | 旅途 | 马赛、地中海航线 |
| 翠绿 `●` | 采风 | 绥远 |
| 深灰 `●` | 长眠 | 香山 |

地图上还有 **彩色路线**：
- 靛蓝 `—` 赴欧航线
- 金色 `—` 回国航线
- 翠绿 `—` 绥远考察路线

**操作**：
- 鼠标悬停标记 → 弹出地点简介
- 点击标记 → 地图飞至该地点，同时右侧面板展开详情
- 鼠标滚轮缩放、拖拽平移

### 5.2 场次导航

地图左侧的 **序 / 壹–玖** 按钮 = 歌剧的 10 个场次。

| 按钮 | 场次 | 地点 | 主要内容 |
|------|------|------|----------|
| 序 | 嗣音有人 | 香山 | 刘小蕙追忆父亲 |
| 壹 | 留别北大 | 北京 | 新文化运动论战、蔡元培送行 |
| 贰 | 雾雨伦敦 | 伦敦 | 伦敦求学、创作「她」字诗 |
| 叁 | 他乡知己 | 伦敦 | 赵元任来访、惺惺相惜 |
| 肆 | 迁渡巴黎 | 巴黎 | 巴黎语音实验、博士学位 |
| 伍 | 东方荣耀 | 巴黎 | 法国国家文学博士颁授 |
| 陆 | 重归北大 | 北京 | 回国任教、建立语音乐律实验室 |
| 柒 | 音律和鸣 | 北京 | 除夕团圆、刘天华演奏《良宵》 |
| 捌 | 绥远采风 | 绥远 | 草原采风、搜集民歌 |
| 玖 | 流风无尽 | 香山 | 英年早逝、蔡元培题碑 |

**点击场次按钮** → 地图飞至对应地点 → 面板展开剧情 → **面板头出现 ♪ 按钮**。

### 5.3 聆听歌剧音频 🔊

这是本作品的核心体验——将歌剧视频中的音频按 10 个场次切分，每个场次均可独立播放。

**操作步骤**：
1. 点击地图左侧的场次按钮（序/壹/贰…）
2. 右侧抽屉面板展开 → 面板头出现一个 **♪ 音符按钮**
3. 点击 ♪ 按钮 → 开始播放该场完整音频
4. 播放中 ♪ 按钮变为 **红色脉冲** 状态
5. 再次点击 → 暂停
6. 点击底部音频条的 **×** → 停止并关闭

**底部音频条**也会同步显示当前播放的场次名称。也可点击音频条的 **▶/⏸** 控制播放。

**10 场总时长**：约 1 小时 40 分钟。

### 5.4 面板五 Tab（点击地点后展开）

右侧抽屉面板包含五个标签页：

| Tab | 内容 |
|-----|------|
| **剧情** | 当前地点关联的歌剧场次剧情摘要 + 「虚实对照」切换按钮 |
| **曲目** | 该场次的歌曲列表，含演唱者、类型、关键唱词 |
| **作品** | 刘半农在该时期/地点的学术著作与文学作品 |
| **史料** | 历史背景补充，与「剧情」对照阅读 |
| **人物** | 该场次的出场人物列表 |

**虚实对照模式**：剧情 Tab 中点击「虚实对照」按钮，脚本内容与历史史实并列显示，方便读者区分戏剧加工与真实历史。

### 5.5 「她」字专题

点击顶部 **「她」** 按钮 → 弹出浮层，展示「她」字的百年历程：

- 1920 年刘半农在伦敦首次将「她」引入新诗
- 1920–1923 年社会争议与论战
- 赵元任谱曲风靡全国
- 鲁迅评价与百年传承

浮层中的大字 **「她」** 可点击 → 直接跳转到地图上伦敦节点。

### 5.6 江阴三杰

点击顶部 **「三杰」** 按钮 → 刘氏三兄弟生平对比卡片：

| 人物 | 领域 | 代表作 |
|------|------|--------|
| 刘半农 | 文学·语言学 | 《扬鞭集》《教我如何不想她》 |
| 刘天华 | 国乐改良 | 《良宵》《病中吟》《光明行》 |
| 刘北茂 | 二胡作曲 | 《汉江潮》《小花鼓》 |

### 5.7 人物关系网络

点击顶部 **「关系」** 按钮 → D3.js 动态力导向图。

以刘半农为中心，辐射新文化运动、现代语言学、国乐改良三大领域共 18 人。节点可拖拽、连线表示关系亲疏。

点击任意节点 → 弹出该人物详细信息。

### 5.8 人物肖像卡

地图 **左下角** 有一张复古卡片。页面加载时自动显示刘半农肖像，点击地图上人物名或关系图中的节点可切换显示其他人。

- **有照片的人物**（11 人）：显示灰度棕褐做旧风格的民国肖像，**鼠标悬停时色彩微微还原**
- **无照片的人物**（4 人）：显示书法剪影占位

肖像卡片列出：姓名、身份、生卒年、与刘半农的关系。

### 5.9 声调实验室

点击顶部 **「声调」** 按钮 → 致敬刘半农 1925 年在巴黎建立的 **「语音乐律实验室」**（中国第一个现代语音学实验室）。

**功能**：
- **参考声调**：点击阴平/阳平/上声/去声按钮，播放对应的合成四声示范音
- **波形可视化**：Canvas 画布实时显示声调曲线
- **录音对比**：允许浏览器使用麦克风，录制你自己的发音并与五度标记法对比
- **五度标记法说明**：折叠面板介绍赵元任发明的这套声调符号系统

### 5.10 搜索功能

顶部搜索框支持按以下维度搜索：
- 地点名称（如「伦敦」）
- 人物姓名（如「蔡元培」）
- 拼音（如「liu」）
- 类别（如「留学」）

输入时实时显示匹配结果，点击结果跳转。

### 5.11 多维筛选

顶部筛选栏提供三个维度交叉过滤：

| 维度 | 选项 |
|------|------|
| **类别** | 故乡与求学、执教、留学、旅途、采风、长眠 |
| **人物** | 15 位出场人物 |
| **时期** | 早期（1891–1917）、欧陆（1917–1925）、晚期（1925–1934） |

点击筛选项 → 地图上不匹配的节点自动隐藏。点击「清除筛选」恢复全部显示。

### 5.12 自动巡游

点击顶部 **「播放」** 按钮 → 自动按场次顺序走完全程。地图依次飞至每个地点，伴随动画路线。

### 5.13 底部时间轴

44 年时间线（1891–1934），标注 12 个关键年份节点。悬停查看事件详情。

### 5.14 「关于」页面

## 六、常见问题

### Q1：打开 start.bat 一闪而过 / 提示找不到 Python

→ **Windows 用户**：确认压缩包已完整解压（不是直接在压缩包内双击）。确保文件夹内有 `runtime\python\python.exe` 文件。
→ **Mac/Linux 用户**：终端输入 `python3 --version` 确认已安装。未安装则 `brew install python3`（Mac）或 `sudo apt install python3`（Ubuntu）。

### Q2：浏览器打开后显示「无法连接」

→ 可能端口 8000 被其他程序占用。
→ 双击 start.bat 重新启动（会自动处理端口冲突）。
→ 或手动指定其他端口：`python server_nocache.py`，然后打开 `http://localhost:8000`。

### Q3：地图一片空白 / 标记不显示

→ 检查网络连接——地图瓦片首次加载需要联网。
→ 按 `Ctrl + Shift + R` 强制刷新浏览器缓存。

### Q4：点击场景音频 ♪ 按钮没有声音

→ 确认浏览器允许音频自动播放（Chrome 默认允许用户主动点击触发的播放）。
→ 检查电脑音量设置。
→ 按 F12 打开控制台，查看是否有红色报错信息。

### Q5：页面显示奇怪/样式错乱

→ 按 `Ctrl + Shift + R` 强制刷新。
→ 如果之前打开过该页面，浏览器可能缓存了旧版本文件。

### Q6：人物肖像不显示

→ 确认 `assets/portraits/` 文件夹存在且内含 11 张图片。
→ 图片加载失败时会自动显示书法剪影占位，不影响浏览。

### Q7：如何关闭服务器

→ **Windows**：关闭标题为「OperaMap Server」的命令行窗口。
→ **Mac/Linux**：在终端中按 `Ctrl + C`。
→ 关闭浏览器页面不会自动停止服务器——记得关掉命令行窗口。

## 八、技术说明

- **地图引擎**：[Leaflet 1.9.4](https://leafletjs.com/) + 高德地图中文瓦片
- **关系图**：[D3.js v7](https://d3js.org/) 力导向布局，支持拖拽与缩放
- **音频**：HTML5 Audio API，MP3 格式；Web Audio API 用于声调合成
- **字体**：Google Fonts — Noto Serif SC（正文）/ Noto Sans SC（UI），自动加载
- **样式**：CSS Custom Properties（变量）驱动的「民国印刷风」冷灰墨韵调色板
- **架构**：IIFE 模块模式 + 自定义事件总线（pub/sub），无框架依赖
- **缓存**：服务器端设置 `Cache-Control: no-store`，确保每次加载最新文件
- 所有外部依赖（Leaflet、D3）均从 CDN 加载，无需 npm install
