# 教我如何不想她 — 刘半农生平地理叙事

交互式地图 · 歌剧伴生数字作品

## 项目简介

《教我如何不想她》是一部关于中国现代语言学家、文学家、新文化运动先驱 **刘半农**（1891–1934）生平事迹的原创歌剧。

本网站是该歌剧的 **「数字伴生作品」**，以交互式地图为核心，串联刘半农在江阴、北京、伦敦、巴黎、绥远等 11 个地点的生平轨迹。

**编剧**：彭峰、张吉义　|　**作曲**：禹永一

## 快速启动

**无需安装任何软件**——解压即用。

| 系统 | 操作 |
|------|------|
| Windows | 双击 `start.bat` |
| Mac / Linux | `chmod +x start.sh && ./start.sh` |
| 手动 | `python server_nocache.py` |

浏览器打开 `http://localhost:8000`。

> Windows 自带 Python 3.12 嵌入式版（`runtime/python/`），无需单独安装。Mac 和 Linux 通常已预装 Python 3。
