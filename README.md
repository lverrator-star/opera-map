# 教我如何不想她 — 刘半农生平地理叙事

交互式地图 · 歌剧伴生数字作品

## 项目简介

《教我如何不想她》是一部关于中国现代语言学家、文学家、新文化运动先驱 **刘半农**（1891–1934）生平事迹的原创歌剧。本网站是这部歌剧的「数字伴生作品」，以交互式地图为核心，将刘半农在不同历史时期的求学、游历、采风地点串联起来，展现其文学作品、学术思想与歌剧脚本内容的地理叙事。

## 功能层次

### ✅ Layer 0：数据基础
- 11 个地点节点（含坐标、历史描述、脚本关联）
- 10 场歌剧（序+9场，含 28 首曲目、台词、人物）
- 4 条旅行路线 GeoJSON

### ✅ Layer 1：MVP 核心
- Leaflet 地图 + 11 个分类标记
- 点击标记 → 右侧抽屉展开（场次内容 + 台词 + 曲目）
- 底部时间轴 1891–1934

### ✅ Layer 2：双视图系统
- 🌍 世界视图 + 🇨🇳 中国视图切换
- 场次纵列导航（序→玖）
- 「播放全程」自动飞行（含路线动画）
- 地图标记脉冲动画

### ✅ Layer 3：内容深度
- 抽屉 5 个 Tab：剧情/曲目/作品/历史/人物
- 「她」字诞生专题页
- 「江阴三杰」兄弟谱系
- 人物关系网络图（纯 CSS + SVG）
- 虚实对照模式（脚本 ↔ 史料）

### ✅ Layer 4：时间轴与叙事
- 12 个关键事件标记（含图标和描述）
- 点击时间轴 → 地图飞行 + 抽屉展开
- 点击地图节点 → 时间轴高亮对应年份段
- 自动播放：沿时间顺序飞行，每场停留 3.5 秒

### ✅ Layer 5：多媒体
- Web Audio API 主题提示音（不同地点不同音色）
- 音频播放条 UI
- 曲目可展开歌词 + 试听提示

### ✅ Layer 6：进阶交互
- 多维度筛选（类别/时期/主题）
- 人物点击 → 关系卡片 + 地图高亮
- 知识图谱浮层
- ESC 键关闭所有浮层/清除筛选

## 技术栈

- **地图**: [Leaflet](https://leafletjs.com/) + CARTO 浅色瓦片
- **样式**: 纯 CSS（民国风：瓷白 #FAF8F5 / 朱砂 #C41E3A / 墨绿 #2F5F4F）
- **字体**: Noto Serif SC（标题）+ Noto Sans SC（正文）
- **音频**: Web Audio API
- **无框架**: 原生 JavaScript（IIFE 模块模式 + 事件总线）
